# 📚 学生课时管理 CRM 系统

一个简洁高效的学生课时管理系统，支持**语音销课**功能，老师只需说一句话就能完成销课操作。

## ✨ 核心功能

### 1. 🎤 语音销课（核心功能）
老师只需说一句话，系统自动识别并完成销课：
- **"张三销2节数学课"** → 自动为张三的数学课扣除2节
- **"李四上了一次英语课"** → 自动为李四的英语课扣除1节
- **"王五增加3节物理课"** → 自动为王五增加3节物理课

### 2. 👥 学生管理
- 添加/删除学生
- 管理多个课程包
- 记录学生联系方式和备注

### 3. 📊 课时管理
- 实时显示每个学生的课时余额
- 快速销课/增加课时
- 完整的操作历史记录

### 4. 🔍 搜索和筛选
- 按学生名字搜索
- 按课时状态筛选（有剩余/已用完）

### 5. 📈 数据统计
- 总学生数
- 本月销课总数
- 今日销课数

## 🚀 快速开始

### 方式一：直接打开 HTML 文件
```bash
# 进入项目目录
cd ~/.qclaw/workspace/crm-student

# 用浏览器打开
open index.html
```

### 方式二：启动本地服务器
```bash
# Python 3
python3 -m http.server 8000

# 然后访问：http://localhost:8000
```

## 📖 使用指南

### 添加学生
1. 点击 **"➕ 添加学生"** 按钮
2. 填写学生信息（姓名、电话、课程包）
3. 点击 **"保存"**

### 语音销课（推荐方式）
1. 点击 **"🎤 语音销课"** 按钮
2. 说出或输入销课命令，例如：
   - "张三销2节数学课"
   - "李四上了一次英语课"
   - "王五增加3节物理课"
3. 系统自动识别并完成操作

### 快速销课
1. 点击学生卡片查看详情
2. 在 **"快速销课"** 区域选择课程
3. 输入销课节数
4. 点击 **"销课"** 按钮

### 查看历史记录
1. 点击学生卡片查看详情
2. 向下滚动查看每个课程的操作历史

## 🎯 语音命令格式

系统支持多种自然语言格式：

| 命令 | 说明 |
|------|------|
| `张三销2节数学课` | 销课 |
| `李四扣1节英语课` | 销课（扣课） |
| `王五上了一次物理课` | 销课（上课） |
| `赵六增加3节语文课` | 增加课时 |
| `孙七加2节化学课` | 增加课时 |

## 💾 数据存储

所有数据自动保存到浏览器的 **LocalStorage**，关闭浏览器后数据不会丢失。

### 导出数据
```javascript
// 在浏览器控制台运行
console.log(JSON.stringify(crm.students, null, 2))
```

### 导入数据
```javascript
// 在浏览器控制台运行
localStorage.setItem('students', JSON.stringify(yourData))
location.reload()
```
